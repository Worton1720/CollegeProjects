# МирИгрушек — веб-приложение (PHP + PostgreSQL)

Магазин игрушек. Подключается к базе `root_db` на PostgreSQL (`127.0.0.1:5433`).

## Запуск

Нужен PHP с расширениями `pdo_pgsql` и `gd`. Из папки `app/`:

```bash
php -S 127.0.0.1:8080
```

Открыть: <http://127.0.0.1:8080/>

Параметры подключения к БД — в `inc/config.php` (по умолчанию `127.0.0.1:5433`,
база `root_db`, пользователь `db_user`, пароль `db_pass`).

## Роли и возможности

- **Гость / клиент** — просмотр списка товаров (без поиска, сортировки, фильтрации).
- **Менеджер** — товары с поиском/сортировкой/фильтром + просмотр заказов.
- **Администратор** — всё вышеперечисленное + добавление/редактирование/удаление товаров и заказов.

## Реализовано по заданию

- Окно входа, гостевой доступ, выход на экран входа, ФИО и роль в правом верхнем углу.
- Список товаров: фото (заглушка `placeholder.png` при отсутствии), подсветка строк
  (скидка > 17% → `#FFDEAD`, нет на складе → голубой), перечёркнутая старая цена при скидке.
- Поиск по всем текстовым полям, сортировка по цене/остатку, фильтр по поставщику
  («Все поставщики» сбрасывает) — в реальном времени, совместно.
- Форма добавления/редактирования товара: выпадающие списки категорий/поставщиков,
  ID только для чтения при редактировании и `max+1` при добавлении, загрузка фото с
  приведением к 300×200 и удалением старого файла, валидация (цена/количество ≥ 0).
- Удаление товара запрещено, если он есть в заказе.
- Заказы: список, форма добавления/редактирования (статус/пункт/клиент/даты + состав),
  удаление. Стиль по Приложению 3 (Arial, фон `#FFFFFF`/`#F5DEB3`, акцент `#DEB887`), логотип и иконка.
