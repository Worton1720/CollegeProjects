# Comfort-Rest Travel Agency Administration System

Система администрирования для турфирмы "Комфорт-отдых". Полнофункциональное веб-приложение для управления турами, клиентами и странами.

**Stack**: PHP 8.3 + SQLite 3 + Vanilla JS + Bootstrap 5

---

## Быстрый старт

### Требования

- **PHP** 8.3+
- **SQLite** 3
- Права на запись в директории `db/` и `logs/`

### Запуск (Development)

```bash
cd comfort-rest
php -S localhost:8000
```

Откройте: `http://localhost:8000/login.html`

**Учетные данные:**

- Логин: `admin`
- Пароль: `Admin@12345`

---

## Возможности

- **Управление турами:** Создание, редактирование, фильтрация по странам.
- **База клиентов:** Учет данных, история поездок, функция "Быстрое добавление" (создание клиента + назначение на тур в один клик).
- **Справочник стран:** Учет визового режима с защитой от случайного удаления (FK constraints).
- **Безопасность:** BCrypt хэширование, сессионная авторизация, защита от SQL Injection.
- **Валидация:** Двойной контур проверки данных (Client-side + Server-side).

---

## Структура проекта

- `/api` — Серверная логика (PHP REST API).
- `/js` — Фронтенд (Vanilla JS, без зависимостей).
- `/db` — База данных SQLite.
- `/logs` — Журналы запросов, ошибок и операций.
